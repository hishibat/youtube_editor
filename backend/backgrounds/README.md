# 背景素材フォルダ

このフォルダに背景素材を配置してください。

## カテゴリ構造

```
backgrounds/
├── funny/       # ネタ系（面白い背景）
├── stylish/     # おしゃれ系
└── realistic/   # 実写系
```

## 対応形式

- 静止画: `.jpg`, `.png`
- 動画: `.mp4`, `.webm`
- アニメーション: `.gif`

## 命名規則

ファイル名がそのまま背景の表示名になります。
アンダースコア `_` はスペースに変換されます。

例: `galaxy_stars.mp4` → "Galaxy Stars"
